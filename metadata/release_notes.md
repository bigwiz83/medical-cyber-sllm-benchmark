# Release notes

Version 1.0.0 is the first public-package candidate for the 60-scenario, ten-condition benchmark. It
contains 3,000 evaluation cells, all article analysis outputs, three figure masters, three main
tables, and the selected supplementary analyses.

Before public distribution, the author must approve separate code and data licenses, confirm
third-party data terms, complete citation identity/article metadata, change CI to run the publication
gate, and regenerate `metadata/release_manifest.json`. No repository publication is authorized by
this staging note.
