# Security and responsible use

This repository is an evaluation artifact for defensive research on public and synthetic data.
It does not authorize scanning, exploitation, persistence, privilege escalation, or access to any
real hospital system. The supplied scripts perform local file validation and statistical analysis
only. They do not issue network requests or execute model-authored commands.

Do not add patient, staff, credential, or real-hospital data. Report an accidental sensitive-data
exposure privately to the repository owner rather than opening a public issue containing the data.
